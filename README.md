# Atividade HTML para Alunos

## Instruções para os Alunos

### Passo 1: Fork do Repositório

1. Clique no botão "Fork" no canto superior direito desta página
2. Confirme o fork para sua conta pessoal

### Passo 2: Clonar o Repositório

```bash
git clone https://github.com/SEU_USUARIO/atividade-html-alunos.git
cd atividade-html-alunos
```
